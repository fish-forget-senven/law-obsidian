# 一、国有经济
- 国有经济是为了促进国民经济的健康发展，由国家（政府）出资，或直接设立国有独资企业，或控股、参股设立企业，并由国家直接或间接管理和经营国有资本的一种经济形式。它是公有制经济最重要的组成部分。作为社会主义市场经济的基础，其不仅数量庞大，而且分布在重要的行业领域，对国民经济发挥着重要作用。国有经济主要有以下几个特征：
	- 第一，国有经济是国家促进国民经济平衡、协调、可持续健康发展的基本方式。中共十五届四中全会《中共中央关于国有企业改革和发展若干重大问题的决定》指出，国有经济是“国家引导、推动、调控经济和社会发展的基本力量”，“国有经济在关系国民经济命脉的重要行业和关键领域占支配地位，支撑、引导和带动整个社会经济的发展，在国家宏观调控目标中发挥重要作用”。国有经济的主导地位和主导作用，主要体现在控制力上。国有经济的主导作用要求国有经济在国民经济运行中必须成为国家宏观调控的重要手段。
	- 第二，国有经济的实现形式多样，既有国有独资企业，又有国有控股企业，还有各类利用国有资本参股的企业。
	- 第三，国家（政府）管理和经营的对象是国有资本而不是国有资产，国家（政府）管理和经营国有资本的最直接目的是保值增值。管理和经营的具体方式可以是直接的，也可以是间接的。国家（政府）在运用这两种方式时，应以间接管理和经营为主，以直接管理和经营为辅。国家（政府）管理和经营要实行两种转变：一是要从管资产转变为管资本；二是从直接管理与经营转变为间接管理与经营。有学者将国有经济划分为企业型、财政型和权力型这三种类型。我们不赞成此种分法。首先，“企业型国有经济”这个概念，强调的是属于国家所有的以企业资产形式存在的财产，同时强调国家直接所有或占有。国家对资产直接所有或占有，并不是最终目的，国有经济存在的基础是资本而不是资产。资本是投入企业用于经营的资产，其本质是追求增值。必须设法采取多种形式，将国有资产转化为资本从事企业的经营活动，为国家和社会谋取更多的利益。其次，“财政型国有经济”这个概念不能成立。国家所有的以非企业资产形式存在的财产，无论是以实物形式存在的资产，还是以货币形式存在的资产，当然属于国有资产。这类资产主要用于维持国家机器的运行，它与投入企业运行的国有资本有很大区别。前者属于消耗型的，一般被称为“公共财政支出”；后者属于经营型的，它能带来资本收入，两者管理的原则、方式方法都是不同的。最后，“权力型国有经济”这个概念也不成立。国家据以组织、指导、管理和控制整个国民经济是国家的经济管理职能，并不能成为一种经济形态。国家对整个国民经济的控制，特别是对非国有经济的控制，不是为了攫取经济利益，而是为了维护社会经济秩序，促进国民经济稳定增长。即使是国家利用强制力对国民收入进行再分配，也是为了保证社会的公平和稳定。“权力型国有经济”容易被理解为“权力经济”，即以权力换取利益的经济，这不是我们所期望的经济形态。
# 二、国有资产
“国有资产”（state—owned assets）由“国有”和“资产”两个词组合而成。“国有”即“国家所有”，体现的是国家这一特殊主体在民法上享有的所有权，即占有、使用、收益和处分的权利。根据《中华人民共和国宪法》第6条的规定，我国社会主义经济制度的基础是生产资料的社会主义公有制，即全民所有制和劳动群众集体所有制。社会主义全民所有制是社会全体成员共同占有社会生产资料的一种所有制形式。这种所有制形式在法律上必然表现为社会主义国家所有权。全体人民只有通过他们的共同代表者国家才能形成一个整体共同支配全民的生产资料，使生产资料在分配使用上从属于全社会的共同利益。如果全民财产的所有权不交由能代表全国的国家来行使，而交由直接占有全民财产的组织或个人来行使，那么全民财产也就不复存在，全民所有制也就不复存在。“资产”（property或asset）是会计学和经济学中的一个重要概念。在会计学上，资产的核心特征有两个：其一，资产是一种经济资源；其二，资产能给控制者带来未来的经济利益。我国2007年实施的《企业会计准则》在第20条对“资产”作了明确规定，即“企业过去的交易或者事项形成的、由企业拥有或者控制的、预期会给企业带来经济利益的资源”。在经济学上，“资产”同样也被界定为能够带来经济利益的资源，它表现为一定时点的财富存量，由一定数量的物质资料和权利构成。与会计学中的资产概念不同的是，经济学上的资产概念还强调“稀缺性”，这是资产的社会属性。在法学界，资产主要体现为财产，资产是可以作为生产要素投入到生产经营过程中并能带来经济利益的财产和财产权利。美国经济学家约翰·康芒斯指出，资产的法律意义是财产，而财产的经济利益是资产。《布莱克法律词典》认为，“财产是指占有、使用和收益的一切外在事物，包括物质财富、非实物形态的债权、知识产权等其他权益”。《牛津法学大词典》认为，“property”即财产，主要有三层含义：（1）被拥有或者可能被拥有的事物，如财富、财务、土地；（2）所有权，即唯一拥有、享有和使用某物的权利；（3）归某人合法所有之物，即受法律保护而私人享有的有形财产权（土地、金钱、货物）和无形财产权（著作权、专利权等）。

- 综上所述，所谓国有资产，在法律意义上是指国家所有的财产，是国家以各种形式对企业投资及收益、接受馈赠形成的，或凭借国家权力取得的，或依据法律认定的各种类型的财产和财产权利。我国的国有资产主要是通过以下四种途径形成：
	1. 国家以各种形式投资所形成的资产（主要指国家投入国有企业及其他企业的用于经营的资本金及其权益）以及国家对行政事业单位拨入经费形成的资产。
	2. 国家接受馈赠形成的资产，包括公民个人赠与国家的财产和外国友人、团体、政府赠与我国的财产。
	3. 国家凭借权力取得的属于国家的资产，包括依法没收的官僚资本和敌伪财产；依法宣布为国有的城镇土地、矿藏、海洋、河流、森林、滩涂和湖泊等自然资源；依法赎买的资本主义工商业；依法征收、征用的房地产；依法没收的其他财产；依法征收和收缴的税、费和罚金；依法认定和接受的无主财产和无继承人的财产等。
	4. 凡在我国境内所有权不明确的各项财产，除法律另有规定者外，也属于国有资产。

- 国有资产具有以下几个特征：
	- 所有者具有唯一性。在现代社会中，唯有国家在法律上是全民的代表，所以，对全民所有的财产可以在法律上充当所有者的只能是国家。
	- 所有权界定具有法定性。即国有资产的各种取得方式都直接由法律规定，并且有的取得方式为国有资产所特有，如凭借国家权力取得。
	- 表现形态具有多样性。即国有资产表现为多种形态的经济资源，既可以是财产，也可以是财产权利；既可以是人造财产，也可以是自然资源；既可以是有形财产，也可以是无形财产。

在现实经济生活中，国有资产有广义和狭义两种理解。广义的国有资产是指所有权属于国家的一切财产，包括经营性国有资产、行政事业性国有资产以及资源性国有资产。其中，经营性国有资产，是指国家作为出资者在企业中依法拥有的资本及其权益，是国有资产中最重要、最活跃的部分，也是国有资产管理的重点对象；行政事业性国有资产，是由行政事业单位占有、使用、在法律上确认为国家所有，能以货币计量的各种经济资源的总和，其中包括：国家拨给行政事业单位的资产、行政事业单位接受捐赠和其他法律确认为国家所有的资产；资源性国有资产是指具有开发价值但尚未投入生产经营领域的国家自然资源，包括土地、矿藏、水流、森林、山岭、草原、滩涂、海洋等。而狭义的国有资产仅指经营性国有资产，主要包括：企业使用的国有资产、行政事业单位占有和使用的非经营性资产中通过各种形式为获取利润转作经营的资产、国有资源中投入生产经营过程的资源。我国2003年通过的《企业国有资产监督管理暂行条例》以及2008年通过的《中华人民共和国企业国有资产法》（以下简称《企业国有资产法》）采用的都是狭义的概念。

《企业国有资产法》在立法中，对于国有资产法的调整范围应当包括哪些国有资产，曾有争议。1993年，列入第八届全国人大常委会立法规划的国资法草案采取“大”国资法概念，调整全部三类国有资产；而提交给第十届全国人大常委会第三十一次会议审议的草案则将调整范围限定于经营性国有资产，排除了行政事业性和资源性国有资产，属于“小”国资法。实际上，关于国有资产法的调整范围，曾经有过大、中、小三种不同意见。“大”国资法提倡者认为，应该将经营性国有资产、行政事业性国有资产和资源性国有资产都包括在内，制定一部完整的《国有资产法》，使各种国有资产有统一的章法可循，国资立法能够切实起到规范国有资产管理、使用并加以有效保护的作用。提倡“中”国资法者认为，该法使用范围应该是经营性国有资产和行政事业性国有资产，因为这两类国有资产彼此交织并可相互转换，且一直以来都是由财政部主管，可一并调整。提倡“小”国资法者则认为，应当制定一部仅调整经营性国有资产的法律，解决当前亟须解决的现实问题。这种意见是对我国关于国家所有权的认识和统筹管理水平低、部门权力和利益难以协调的现实的一种妥协，认为寄希望于一部法律解决国有资产存在的问题不现实，制定难度大，可操作性也不强。立法机关在汇集了各方意见后，采用了“小”国资法的意见，认为各类国有资产在功能、监管方式等方面有很大不同，难以用一部“大而全”的法律全面调整。我国对资源性国有资产已经有了《中华人民共和国土地管理法》、《中华人民共和国矿山资源法》、《中华人民共和国水法》《中华人民共和国森林法》、《中华人民共和国煤炭法》等相关的专门法律予以调整，行政事业性国有资产的管理方面也有《行政单位国有资产管理暂行办法》、《事业单位国有资产管理暂行办法》等行政规章。目前，最需要专门立法的是国家出资企业中的国有资产，以便维护企业国有资产的合法权益，保障国有资产安全，促进国有资产的保值增值。因而，立法机关决定，先制定一部适用于经营性国有资产，即企业国有资产的法律。
# 三、国有资本
资本是一种生产要素，与其他要素结合能够给所有者带来价值增值，实现所有者的投资利益最大化。所有者对投资利益拥有支配权。资本具有增值性、逐利性、运动性和多样性的特点。国有资本是按照一定经济基础的要求，通过不同途径参与资本运动的国有资产。国有资产不等于国有资本，但国有资产可以转化为国有资本。国有资本的存量、布局和营利能力在一定程度上反映了一个国家国有经济的整体实力和对整个国民经济的控制力。

国有资本具有资本的一般属性，但同时又具有自身的特征，即不以追求利润为唯一目标。由于所有权归属的特殊性，即“公有”，使得国有资本在一定程度上突破了资本本身固有的利润目标的局限。除要保值增值外，还要实现一定的社会公共目标和政治目标，而且，往往是后者优于前者。正是由于国有资本具有这两重属性，因而其一部分职能是生产社会公共产品，例如国防建设、基础设施建设等；另外一部分职能则是在市场经济活动中作为一般资本参与谋取利润的经济活动。前一职能的公有性质体现在为国民经济提供“外部性”条件上，后一职能的公有性质体现在收益的归属上。不论哪一种职能，国有资本的产权性质都没有发生任何变化。

总而言之，国有资本是国家宏观调控的重要工具，是政府影响和干预经济的重要手段，这使得其在国民经济中扮演着其他资本无法替代的角色。
# 四、国有企业
- 国有企业虽在各国都存在，但对什么是国有企业，各国的认定却并不一致。在美国，“联邦公司（即国有企业）是指由联邦政府全部或部分拥有和控制或者虽非联邦政府拥有、但由联邦政府特许资助、并受其控制或监督的组织”。在英国，国有企业必须符合以下三个条件：
	1. 这些工业完全归国家所有或者国家拥有能够对它实现控制的足够股份；
	2. 这些工业收入的大部分不是依靠国会直接拨款或财政部的补贴；
	3. 它们由董事会进行管理。在瑞典、芬兰、巴西、墨西哥等国，凡国家资本占企业资本超过50％的所有企业都被称为国有企业。就国际组织的规定而言，联合国有关部门认为，国有企业是公共所有或受公共控制的公共股份有限公司。

在我国法律中，并没有关于国有企业的准确定义，只有《全民所有制工业企业法》在一定程度上表现出了从法律层面界定国有企业的意图。该法第2条第1款规定：“全民所有制工业企业是依法自主经营、自负盈亏、独立核算的社会主义商品生产和经营单位。”实际上，这个规定对于国有企业的认定仍不清晰。在法学界，代表性的观点认为，国有企业是指“国家或者政府可以根据资本联系，对其实施控制或控制性影响的各种企业”。依据该界定，国有企业包括全民所有制企业（国有独资企业）、国有独资公司、国有控股的有限责任公司或股份有限公司。有学者将其概括为“国有企业是指国家拥有、经营或者控制的生产经营单位”。

从以上国内外的代表性观点可以看出，在对国有企业的内涵界定中，国家与企业间的支配性资本联系以及国家对企业的控制性地位均被特别强调。基于此，我们认为，国有企业应被界定为：企业资本全部或者部分属于国家所有并为国家所控制的企业。对“国有企业”的这一界定，可以视为对国有资产的监管从“实物监管”向“资本监管”的转变，使得国家的角色从实体资产的直接所有权人转变为国有资本所有者，从而带来国有资产监管体制、机制和功能的一系列变化。
依据西方经典经济学理论，国有企业的存在是政府职能的延伸，是政府为了实现某些社会政策目标而投资的特殊企业。由于该类企业主要是为了弥补市场缺陷，缓解市场经济自身发展中的矛盾，因而，其所肩负的功能“天然”地不具有营利性，其经营活动也是以社会整体效益为指导，将社会公益目标居于优先地位。在现实中，它主要存在于一些市场机制所不能及的、非国有资本不愿意投资的某些超越利润之外的，但又是国民经济长远发展和国民生活所必需的行业和领域。对于竞争性、营利性行业，国有企业一般不进入。但在我国，国有企业的存在除上述原因外，还基于社会主义本质所要求的国有经济的控制力。这种控制力主要就是通过国有企业对国民经济活动中关键行业和重要领域的决定性影响和主导作用来实现。

从理论上分析，国有企业的性质应同时有公益性和营利性两个特征。国有企业的产生是为了提供人民生产生活所需的市场不能提供的公共产品或服务，因而具有公益性；但同时，传统“企业”的本质和“资本”的逐利，又使其具有营利性。因而，其同时具有着“准政治人”和“经济人”的双重身份。有学者曾指出，国有企业的这种两重性是相辅相成、互相兼顾的。如果把国有企业仅仅定位于公益性，则会使一些企业不思进取，以至于不断亏损，这样就不会给国家带来收益，反而成了国家的一个沉重负担。同样，如果完全把国有企业定性为营利性，那么那些投资回收期长、投资额大、社会化程度高、经济效益又不显著甚至亏损的行业就没有人来投资。因而，在决定国有资产的投资方向和投资额度时，就不能忽视国有企业的营利性和公益性这两个双重性特点。既要最大限度地谋求营利，又不损及公共的利益和其他任何主体的利益。当然，营利性与公益性两相比较，公益性更为重要，因为国家投资追求营利，最终是为了谋求公益。因此，当营利性和公益性这两个目标发生冲突时，国企应毫不犹豫地放弃营利而服从公共利益，以保障市场机制作用的有效性。

应当说，对国有企业的这种性质分析，在理论上是没有问题的。但现实中，面临的难题是，在同一经济组织体内，同时强调这两种功能，势必会出现冲突，结果反而会造成国有企业的无所适从。[^1]也正是基于该认识，有学者将上述思想进行了发展和延伸，提出了“分类管理”的思路。即按照经营性国有资产生产产品和提供服务的性质不同，将国有企业分为两大类：对于从事商业服务和商品生产的国有企业，应当以经济功能为其主要任务，保值增值是其首要目的，在性质上将其定位于“商事企业”的私法人；而对于生产公共产品和提供公共服务的国有企业，则应以社会功能为根本追求，在性质上将其定位为“公共企业”的特殊法人。

[^1]:这一点，我国原来的银行业有过教训。在政策性银行建立之前，我国商业银行即面临了“经济效益”与“社会效益＂难以兼容的尴尬，结果造成商业银行经营效率的低下。
# 五、国有经济、国有资产、国有资本与国有企业相互关系
国有经济是一个世界性的经济现象，它的出现是社会化大生产的产物。在社会主义国家，国有经济大都处于国民经济的绝对支配地位；在发达的资本主义国家和发展中国家，尽管各国国情不同，社会制度和经济制度也存在着差异，但都拥有一定比例的国有经济成分。国有经济本身并没有固定不变的社会属性，它的属性取决于利用它的国家的社会属性。我国是社会主义国家，国有经济是公有制经济的主体，服务于全民。

国有资产与国有经济是两个相互联系又有区别的概念。国有资产侧重于从静态角度，从归属关系方面描述特殊的民事主体—国家所有的财产和财产权利。国有经济则侧重于从宏观角度，从经济形式方面描述一种与私有经济相对应的经济现象。尽管两者的角度、侧重点有所不同，但指向基本一致。国有资产是国有经济的前提和基础，没有国有资产就没有国有经济。

国有资本与国有经济是两个既密切联系又有区别的概念。国家（政府）对企业投资形成国有资本，国有经济的运行表现为国有资本的运作，或者说通过国有资本的运作来实现国有经济。作为国有经济基础的国有资本，不是泛指国家（政府）所有并由国家行使所有权的财产，而有其特指的内涵。与国有经济相联系的国有资本是指由国家出资、占有，作为生产要素用于生产经营，并在经营中获得收益以及相应权益的资产。因此，与国有经济相联系的国有资本不包括政府机关及非经营性事业单位的国有资产，也不包括国家所有的未开发的自然资源，而是专指国有的生产经营性资产（资本）。

作为国有资本的主要载体和国有经济主要组织形式的国有企业，则是国有经济的微观经济组织，是国有经济的主要实现形式。今后，从国有经济的分布领域来看，国有企业存在的领域应当逐步缩小到涉及国家安全的行业、提供重要公共产品和服务的行业以及自然垄断领域中的管道设施行业。此外，为推动技术进步，国家可以投资设立支柱产业和高新技术产业中的重要骨干企业。

国有企业和国有经济是两个相联又不同的概念。前者是指国有经济的微观经济形式，是财产组织形式和经营方式，也是国有经济的主要载体。同时，国有经济的效率及质量提高也依赖于国有企业管理水平和竞争力的提升。传统的单一投资主体的国有企业制度是计划经济体制的微观经济组织形式，它与市场经济体制不相适应，必须进行改革。建立投资主体多元的、所有者和经营者相分离的有独立经济利益的公司制（法人）企业是今后国有企业改革的方向。在市场经济体制下，传统的国有企业形式将被“公司制企业”所取代，通过改变传统的国有财产的组织形式和经营方式，转变经营机制，使国有经济更有效地发挥主导作用。但同时，一些经营效益差的国有企业也将被淘汰。此外，随着国有企业改革的推进，国有独资企业只保留极少数，国有控股企业也不可能太多，多数是国有参股企业。因而改革的结果将是“国有企业”数量的绝对减少，但国有经济随着国有资本的积累和国家投资的增加，其资本规模将不断扩大，主导作用会不断增强。
