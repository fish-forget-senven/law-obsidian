# 诉讼标的[^1]★★★（按简答掌握）
- 诉讼标的[^2]，是指当事人之间争议的并要求人民法院作出裁判的权利义务关系。其基本功能包括：
	1. 确定当事人攻击防御重心的依据。
	2. 确定法院审理与裁判范围的依据。
	3. 确认是否构成重复起诉的依据。
	4. 判断是否构成客观的诉的合并的依据。
	5. 作为判断是否构成诉的变更的依据。
	6. 作为确定既判力客观范围的依据。

诉权，民事主体的宪法权利，是民事主体启动民事诉讼程序的前提和基础。
