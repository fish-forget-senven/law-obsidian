# 裁判观点
1. 只在以下两种情形时可以认定非法用工关系：第一，无营业执照或者未经依法登记、备案的单位以及被依法吊销营业执照或者撤销登记、备案的单位雇用劳动者进行劳动；第二，有用工主体资格的用人单位使用童工。
2.  非法用工关系属于独立的法律关系类型，其既不构成劳动关系，也不构成劳务关系。
3. 非法用工关系下，劳动者已经付出劳动的，该单位或者其出资人应当依照《劳动合同法》有关规定向劳动者支付劳动报酬、经济补偿、赔偿金；给劳动者造成损害的，应当承担赔偿责任。
4. 非法用工情形下人员伤亡的，不适用《工伤保险条例》进行赔偿，而应当根据《非法用工单位伤亡人员一次性赔偿办法》由非法用工单位向伤亡人员或其近亲属进行赔偿。
# 一、非法用工的概念和构成
非法用工的概念来源于原劳动和社会保障部于2003年9月23日颁布、2004年1月1日开始实施的《非法用工单位伤亡人员一次性赔偿办法》[^1]，该赔偿办法[[../../../法律法规汇编/劳动法/劳动争议相关/非法用工单位伤亡人员一次性赔偿办法#^ukuajj|第2条]]只规定了“非法用工单位伤亡人员”的概念，并未直接对非法用工的概念作出界定，但是司法实践普遍据此规定认为非法用工是指没有用工主体资格的单位雇用劳动者劳动或者用人单位使用童工的用工情况。为了方便理解，可以将非法用工关系看成一种在主体资格方面有致命缺陷的、不被法律认可的“劳动关系”，这一点对于区分非法用工关系与劳务关系至关重要。

- 非法用工包括两种情形：
	- 第一，无营业执照或者未经依法登记、备案的单位以及被依法吊销营业执照或者撤销登记、备案的单位雇用劳动者进行劳动。此种情况下，单位不具备用工主体资格。
	- 第二，用人单位使用童工。在此种情况下，用人单位虽然具有用工主体资格，但是劳动者却是童工，使用童工是被各国法律及国际公约所禁止的。

[^1]:上述赔偿办法于2010年12月31日被修订，2011年1月1日正式实施，第2条没有变动。
# 二、非法用工关系的定性
## （一）缺乏合法经营资格的非法用工单位与劳动者之间的关系
- 对于此类单位与劳动者之间形成的关系是否为劳动关系、是否为有效劳动合同，观点不一致。
	- 第一种意见认为此类用工关系应属劳动关系，
	- 第二种意见认为此类主体不能构成劳动法上的用人单位，其与所雇用的人员所签订的劳动合同为无效合同，双方不能形成劳动合同关系。

- 最高人民法院倾向于第二种意见[^2]，笔者也赞同，但是笔者认为还可以作进一步划分：
	1. 自始不具备合法经营资格的用人单位雇用劳动者劳动的，如果双方关系已经具备除主体资格外的其他劳动关系特征的，构成非法用工关系，单位与劳动者之间不能形成劳动关系或者劳务关系。
	2. 用人单位被吊销营业执照之前已经与劳动者建立劳动关系的，营业执照被吊销后，根据[[../../../法律法规汇编/劳动法/劳动争议相关/中华人民共和国劳动合同法#^dhakod|《劳动合同法》第44条]]的规定，该单位与劳动者之间的劳动合同终止，双方的劳动关系也相应终止。如果该单位仍然延续之前的用人模式继续使用原劳动者，则双方构成非法用工关系。如果用人单位在营业执照被吊销前已经与劳动者构成劳务关系的，则双方法律关系定性不受营业执照是否被吊销的影响。

[^2]:最高人民法院民事审判第一庭编著：《最高人民法院劳动争议司法解释（三）的理解与适用》，人民法院出版社2010年版，第70～71页。
## （二）用人单位与童工之间的关系
用人单位雇用童工劳动的，除极个别的法定情形外，双方均不成立包括劳动关系在内的合法的法律关系，双方只能成立非法用工关系。
# 三、非法用工的法律后果及法律责任
因为非法用工关系是一种在主体资格方面有致命缺陷的“劳动关系”，在法律上不属于劳动关系，所以一方面劳动关系双方无法享受到完全的劳动法权利，另一方面在劳动报酬、工伤待遇等事关劳动者核心权益方面，法律规定相应待遇水平不能低于正常劳动关系项下的保护水平。在程序方面，非法用工发生的争议被纳入劳动争议范畴处理。具体如下：
## （一）劳动报酬、伤害待遇方面不低于劳动关系项下保护水平
1. 根据[[../../../法律法规汇编/劳动法/劳动争议相关/中华人民共和国劳动合同法#^x5vse1|《劳动合同法》第93条]]的规定，对不具备合法经营资格的用人单位的违法犯罪行为，依法追究法律责任；劳动者已经付出劳动的，该单位或者其出资人应当依照《劳动合同法》有关规定向劳动者支付劳动报酬、经济补偿、赔偿金；给劳动者造成损害的，应当承担赔偿责任。
2. 根据[[../../../法律法规汇编/劳动法/工伤/工伤保险条例#^4q5k63|《工伤保险条例》第66条]]的规定，无营业执照或者未经依法登记、备案的单位以及被依法吊销营业执照或者撤销登记、备案的单位的职工受到事故伤害或者患职业病的，由该单位向伤残职工或者死亡职工的近亲属给予一次性赔偿，赔偿标准不得低于本条例规定的工伤保险待遇；用人单位不得使用童工，用人单位使用童工造成童工伤残、死亡的，由该单位向童工或者童工的近亲属给予一次性赔偿，赔偿标准不得低于本条例规定的工伤保险待遇。
3. 根据[[../../../法律法规汇编/劳动法/劳动争议相关/非法用工单位伤亡人员一次性赔偿办法#^aczshy|《非法用工单位伤亡人员一次性赔偿办法》第4条]]的规定，职工或童工受到事故伤害或者患职业病，在劳动能力鉴定之前进行治疗期间的生活费按照统筹地区上年度职工月平均工资标准确定，医疗费、护理费、住院期间的伙食补助费以及所需的交通费等费用按照《工伤保险条例》规定的标准和范围确定，并全部由伤残职工或童工所在单位支付。
## （二）不能享有的劳动法律制度下的部分权利
劳动者要求非法用工单位承担未签订书面劳动合同二倍工资、未订立无固定期限劳动合同二倍工资、未休年休假工资等，不能予以支持。
## （三）非法用工发生的赔偿争议按照劳动争议处理
[[../../../法律法规汇编/劳动法/工伤/工伤保险条例#^4q5k63|《工伤保险条例》第66条]]第2款规定：“前款规定的伤残职工或者死亡职工的近亲属就赔偿数额与单位发生争议的，以及前款规定的童工或者童工的近亲属就赔偿数额与单位发生争议的，按照处理劳动争议的有关规定处理。”[[../../../法律法规汇编/劳动法/劳动争议相关/非法用工单位伤亡人员一次性赔偿办法#^yz7vwn|《非法用工单位伤亡人员一次性赔偿办法》第8条]]规定：“伤残职工或者死亡职工的近亲属、伤残童工或者死亡童工的近亲属就赔偿数额与单位发生争议的，按照劳动争议处理的有关规定处理。”据此，因非法用工发生人员伤残或者死亡的，因赔偿数额发生争议的，按照劳动争议处理。
## （四）非法用工单位出资人的法律责任
[[../../../法律法规汇编/劳动法/劳动争议相关/中华人民共和国劳动合同法#^x5vse1|《劳动合同法》第93条]]规定：“对不具备合法经营资格的用人单位的违法犯罪行为，依法追究法律责任；劳动者已经付出劳动的，该单位或者其出资人应当依照本法有关规定向劳动者支付劳动报酬、经济补偿、赔偿金；给劳动者造成损害的，应当承担赔偿责任。”该条款已经明确了此类用工主体的出资人对劳动者负有支付劳动报酬、经济补偿、赔偿金的责任。[[../../../法律法规汇编/劳动法/劳动争议相关/最高人民法院关于审理劳动争议案件适用法律问题的解释（一）（2021.1.1）#^j5ajku|《新劳动争议解释一》第29条]]规定：“劳动者与未办理营业执照、营业执照被吊销或者营业期限届满仍继续经营的用人单位发生争议的，应当将用人单位或者其出资人列为当事人。”该条款虽然只是从程序角度阐述了当事人诉讼地位问题，但是结合《劳动合同法》第93条的规定可知，司法解释的该条款也蕴含了此类用工主体的投资人的实体法律责任。在司法实践中，也有大量判决判令此类用工主体的投资人对劳动者承担实体责任。
# 四、非法用工单位伤亡人员的赔偿范围
由于非法用工关系不属于劳动关系，单位无法为劳动者办理参加工伤保险的手续，一旦在工作中出现人员伤亡，劳动者无法通过工伤保险制度获得赔偿。为了保护此类劳动者的权益，人力资源和社会保障部根据[[../../../法律法规汇编/劳动法/工伤/工伤保险条例#^4q5k63|《工伤保险条例》第66条]]第2款的授权制定了[[../../../法律法规汇编/劳动法/劳动争议相关/非法用工单位伤亡人员一次性赔偿办法|《非法用工单位伤亡人员一次性赔偿办法》]]。根据该办法的规定，非法用工单位必须按照该办法的规定向伤残职工或者死亡职工的近亲属、伤残童工或者死亡童工的近亲属给予一次性赔偿。具体标准和范围详述如下。
## （一）支付主体
非法用工单位是唯一支付主体。非法用工单位拒不支付一次性赔偿的，伤残职工或者死亡职工的近亲属、伤残童工或者死亡童工的近亲属可以向人力资源和社会保障行政部门举报。经查证属实的，人力资源和社会保障行政部门应当责令该单位限期改正。
## （二）赔偿范围
### 1.劳动者受到伤害或者患职业病的情形
1. 治疗期间的费用：劳动能力鉴定之前进行治疗期间的生活费按照统筹地区上年度职工月平均工资标准确定，医疗费、护理费、住院期间的伙食补助费以及所需的交通费等费用按照《工伤保险条例》规定的标准和范围确定。 ^o19jzq
2. [[../../../法律法规汇编/劳动法/劳动争议相关/非法用工单位伤亡人员一次性赔偿办法#^rzswrt|一次性赔偿金]]：一次性赔偿金数额应当在受到事故伤害或者患职业病的职工或童工经劳动能力鉴定后确定。
	1. 一级伤残的为赔偿基数[^3]的16倍，
	2. 二级伤残的为赔偿基数的14倍，
	3. 三级伤残的为赔偿基数的12倍，
	4. 四级伤残的为赔偿基数的10倍，
	5. 五级伤残的为赔偿基数的8倍，
	6. 六级伤残的为赔偿基数的6倍，
	7. 七级伤残的为赔偿基数的4倍，
	8. 八级伤残的为赔偿基数的3倍，
	9. 九级伤残的为赔偿基数的2倍，
	10. 十级伤残的为赔偿基数的1倍。
3. 劳动能力鉴定费：一次性赔偿金的发放前提是劳动者经劳动能力鉴定为伤残的，所以只要经过劳动能力鉴定的，所发生的劳动能力鉴定费就由非法用工单位支付。
4. 残疾辅助器具费：该费用在《非法用工单位伤亡人员一次性赔偿办法》中并未规定，但是在实践中伤者经常需要配备残疾辅助器具，而辅助器具会因伤者身体情况变化而定期更换，配备及更换的费用一般都较高，该费用是否包含在一次性赔偿金中是一个实践中的难点问题。从该赔偿办法的条文含义看，配备残疾器具辅助费应该已经包含在一次性赔偿金中，定期更换的费用则没有明确规定。从相反解释角度看，定期更换的费用也应当包含在一次性赔偿金中，但是往往因为更换费用较高，会给人一种不公平的感觉。应该说，这一问题在一定程度上体现了《非法用工单位伤亡人员一次性赔偿办法》的局限性，将来彻底解决需要通过修改该赔偿办法的方式来完成。就个案而言，笔者认为可以遵循法律规定的非法用工人员伤亡待遇不得低于《工伤保险条例》所规定的对合法劳动关系中劳动者保护水平的原则，结合具体案情酌情处理。[^4]

[^3]:赔偿基数，是指非法用工单位所在工伤保险统筹地区上年度职工年平均工资。
[^4]:实践中笔者参与处理的一起此类案件就采取了这个原则。
### 2.劳动者死亡的情形
1. 治疗期间的费用：劳动者在治疗期间死亡的，如果有治疗费用发生的，均应由非法用工单位支付，[[理论/实务书籍/劳动人事争议裁判规则/劳动关系的确认/非法用工问题#^o19jzq|标准同前]]。
2. 一次性赔偿金：按照死亡时上一年度全国城镇居民人均可支配收入的20倍支付一次性赔偿金。
3. 一次性丧葬补助：按照死亡时上一年度全国城镇居民人均可支配收入的10倍一次性支付丧葬补助等其他赔偿金。
4. 劳动能力鉴定费：如果劳动者在劳动能力鉴定期间死亡的，如已经发生劳动能力鉴定费，则该费用由非法用工单位支付。
# 裁审要点
1. 非法用工关系不属于劳动关系，而是一种独立的法律关系、为了解决特定违法用工情形下的劳动报酬，特别是伤亡补偿的制度安排，所以在裁判中不能认定其为劳动关系或者劳务关系。
2. 劳动者提供劳动的，应当依照[[../../../法律法规汇编/劳动法/劳动争议相关/中华人民共和国劳动合同法#^x5vse1|《劳动合同法》第93条]]的规定，判令非法用工单位依照《劳动合同法》有关规定向劳动者支付劳动报酬、经济补偿、赔偿金；给劳动者造成损害的，应当承担赔偿责任。
3. 基于[[../../../法律法规汇编/劳动法/劳动争议相关/非法用工单位伤亡人员一次性赔偿办法|《非法用工单位伤亡人员一次性赔偿办法》]]判令非法用工单位向伤亡人员支付的赔偿原则上属于一次性、包干性赔偿。
# 实操指引
1. 有合法用工主体资格的用人单位在招用劳动者时，务必仔细核查劳动者身份证明，防止劳动者通过冒名顶替、虚报年龄等方式进入用人单位提供劳动，从而杜绝用人单位使用童工的风险。
2. 无营业执照或者未经依法登记、备案的单位以及被依法吊销营业执照或者撤销登记、备案的单位为了完成某些特定事项需要用工的，应当以劳务关系或者承揽关系为内容建立双方法律关系。
