# 裁判观点
1. 农民工依照《保障农民工工资支付条例》（本节正文简称《条例》）的规定，向建设单位、施工总承包单位主张清偿工资的，建设单位、施工总承包单位应当依法清偿，清偿后可以在法定情形下向直接招用农民工的个人或者单位追偿。
2. 仲裁和诉讼程序中，农民工主张用人单位应当提供3年的工资发放记录，用人单位不能提供的，用人单位承担不利后果。
# 一、《保障农民工工资支付条例》的意义和作用
## （一）首次单独以农民工为对象制定行政法规
### 1.单独立法意义重大
在《条例》颁布前，我国仅对妇女儿童、老年人、残疾人等特殊人群单独立法，以体现法律对其特殊保护。《条例》颁布后，农民工成为得到行政法规确认的单独特殊保护的群体，农民工也上升为法律概念，意义重大。但是，这种立法带有身份色彩，具体在实践中能够发挥出何种规范和引导作用，有待观察。
### 2.首次明确农民工概念和界定方法
农民工的概念在《条例》实施前只是一种非正式的叫法。《条例》首次对农民工的概念进行了界定，是指为用人单位提供劳动的农村居民。这一概念可以解决大多数农民工身份界定和法律适用问题，但是仍然存在需要进一步明确的领域，最主要的是如何界定形式上已经取消农村户籍地区的居民的务工身份。
### 3.工资支付领域效力级别最高
迄今为止，我国在工资支付领域中就单行规范性文件效力而言，效力最高的只是1995年原劳动部《工资支付暂行规定》。《条例》的颁布将农民工工资保障的规定上升为行政法规，已经超过了对其他任何劳动群体的工资保障力度。
## （二）多部门联动保障力度空前
### 1.多部门协同保障农民工工资支付
根据《条例》规定，多达20个部门从各自职能领域出发协同保障农民工工资支付。例如，人力资源和社会保障部门负责组织协调、管理指导和农民工工资支付情况的监督检查，查处欠薪案件；住房城乡建设部门、交通运输部门、水利部门等负责督办因违法发包、转包、违法分包、挂靠、拖欠工程款等导致的欠薪案件；发展改革部门负责政府投资项目的审批管理，依法审查政府投资项目的资金来源和筹措方式，按规定及时安排政府投资，加强社会信用体系建设，组织对拖欠农民工工资失信联合惩戒对象依法依规予以限制和惩戒；财政部门负责政府投资资金的预算管理，根据经批准的预算按规定及时足额拨付政府投资资金；公安机关负责及时受理、侦办涉嫌拒不支付劳动报酬刑事案件，依法处置因农民工工资拖欠引发的社会治安案件；等等。
### 2.赋予职能部门更多职权
《条例》强调和赋予了职能部门更多职权，以便从行政监管、信用体系、司法衔接等方面全面保障农民工工资支付。例如，人力资源和社会保障部门享有向金融、登记部门查询账户、车辆、房产的权力，而且人力资源和社会保障部门责令支付工资的决定可以直接申请人民法院执行，直接具有执行力。此外，《条例》建立了拖欠农民工工资“失信联合惩戒对象名单制度”，在政府资金支持、政府采购、招投标、融资贷款、市场准入、税收优惠、评优评先、交通出行等方面依法依规予以限制。对于建设单位不提供担保、政府项目拖欠工程款的则纳入征信记录管理。
## （三）全面强化农民工工资保障的监管体系
### 1.全面强化市场主体对农民工工资支付的“自力监管”全面强化“自力监管”主要体现在以下五个方面：
1. 施工总承包单位对分包单位劳动用工实施全方位监督管理，掌握分包单位用工、工资发放等情况，分包单位必须配合（第28条）。
2. 金融机构日常管理农民工工资专用账户（总承包单位开立），发现资金未按约定拨付等情况的，及时通知总承包单位，由施工总承包单位报告人力资源社会保障行政部门和相关行业工程建设主管部门，并纳入欠薪预警系统（第27条）。
3. 建设单位监督总承包单位及时足额支付工资，以项目为单位建立保障农民工工资支付协调机制和工资拖欠预防机制（第29条）。
4. 建设单位、总承包单位联合处理农民工集体讨薪，并报告人力资源和社会保障部门和行业主管部门（第29条）。
5. 分包单位工资支付表经农民工本人签字确认后与当月工程进度等情况一并交施工总承包单位。总承包单位据此通过农民工工资专户发放工资，并交分包单位发放凭证（第31条）。
### 2.新增和强化工资支付保证（金）制度
- 工资支付保证金制度在《条例》颁布之前早已在各地以各种形式存在，体现出较好的保障农民工工资支付的作用，此次《条例》予以系统总结并明确规定，主要有三方面内容：
	1. 建设单位需要向施工单位提供工程款支付担保（第24条）。
	2. 施工总承包单位依照有关规定开设农民工工资专用账户（第26条）。
	3. 施工总承包单位应当按照有关规定存储工资保证金，保证金可以用金融机构保函代替（第32条）。
### 3.全面强化市场主体的行政责任
《条例》共计64条，用了10个条款全方位规定市场主体的行政责任，占比近16％，体现出《条例》对保障农民工工资的空前力度。
## （四）全面加强农民工工资支付的保障体系
### 1.财产保全为保障农民工工资“让路”
《条例》第33条规定，除法律另有规定外，农民工工资专用账户资金和工资保证金不得因支付为本项目提供劳动的农民工工资之外的原因被查封、冻结或者划拨。这一规定，限制了人民法院查封权利的对象范围，目的在于保障农民工工资支付。为落实上述规定，2020年12月25日开始施行《最高人民法院、人力资源社会保障部、中国银保监会关于做好防止农民工工资专用账户资金和工资保证金被查封、冻结或者划拨有关工作的通知》（人社部发〔2020〕93号），对查封、冻结或者划拨上述两类账户进行了细化规定。
### 2.通过信用体系从根源治理欠薪
《条例》第48条规定，用人单位欠薪严重的，单位及法定代表人、直接责任人列入联合惩戒名单，在各种经营中受到资格限制。第49条还规定，建设单位拖欠工程款或者政府投资不到位导致欠薪，建设单位欠薪情况纳入信用体系。
# 二、《保障农民工工资支付条例》对劳动用工制度的影响
## （一）工资支付台账
- 《条例》对农民工工资支付台账进行了特殊规定，体现在两个方面：
	- 第一，延长了保存期限。《条例》第15条第1款规定：“用人单位应当按照工资支付周期编制书面工资支付台账，并至少保存3年。”保存期限超过原劳动部《工资支付暂行规定》所明确的至少保存2年的规定。
	- 第二，工资支付台账记载内容多达13项。《条例》第15条第2款规定：“书面工资支付台账应当包括用人单位名称，支付周期，支付日期，支付对象姓名、身份证号码、联系方式，工作时间，应发工资项目及数额，代扣、代缴、扣除项目和数额，实发工资数额，银行代发工资凭证或者农民工签字等内容。”之前，并未有规范性文件规定工资支付台账应当包含的具体内容。
## （二）增加农民工工资清偿主体，减少工资支付中间环节
- 根据《条例》第18条至第22条的规定，农民工工资支付主体扩展为5个：使用违法派遣方式的用工单位、被借照单位（被借用资质单位）、自然人企业和个体经济组织的出资人、承继单位、履行注销手续的用人单位的主要投资人。其中，用工单位与被借照单位在支付完工资后可以追偿。具体如下：
	1. 用工单位使用个人、不具备合法经营资格的单位或者未依法取得劳务派遣许可证的单位派遣的农民工，拖欠农民工工资的，由用工单位清偿，可向违法派遣的个人或者单位追偿（第18条）。
	2. 被借照单位允许个人或无资质单位借用资质对外经营，导致拖欠所招用农民工工资的，由被借照单位清偿，并可以依法向实际招用农民工的借照单位或者个人追偿（第19条）。
	3. 合伙企业、个人独资企业、个体经济组织等用人单位拖欠农民工工资的，应当依法予以清偿；不清偿的，由出资人依法清偿（第20条）。
	4. 用人单位合并分立前，其清偿欠付农民工的工资；用人单位与农民工书面协商一致的，方可以由用人单位合并分立后的承继单位支付（第21条）。
	5. 用人单位注销前（破产除外），由其清偿农民工工资；该用人单位注销前未清偿的，其主要投资人在注册新用人单位前清偿工资（第22条）。
## （三）建筑施工领域实行特殊规则对农民工工资的影响
从劳动关系角度而言，用人单位对于非己方劳动关系人员不负有支付或清偿工资的义务。但是，《条例》对于建设施工领域作了特殊规定，建设单位、施工总承包单位在法定情形下对于非己方招用的农民工的工资也负有清偿责任。
### 1.建设单位清偿工资的情形
1. 建设单位逾期支付工程款导致拖欠农民工工资的，建设单位应在未结清工程款范围内承担清偿工资责任（第29条）。
2. 建设单位将工程发包给个人或者无合法经营资质的单位，导致拖欠农民工工资的，建设单位负有清偿工资责任（第36条）。
3. 工程建设项目违反国土空间规划、工程建设等法律法规，导致拖欠农民工工资的，由建设单位清偿（第37条）。
### 2.施工总承包单位清偿工资的情形
#### （1）施工总承包单位可以追偿
- 根据《条例》第30条第3款、第4款的规定，施工总承包单位对分包单位、转包单位拖欠农民工工资的，负有清偿责任，但可以依法清偿：
	- 第一，合法分包情形下，分包单位拖欠农民工工资的，由施工总承包单位先行清偿，再依法进行追偿。
	- 第二，工程建设项目转包，拖欠农民工工资的，由施工总承包单位先行清偿，再依法进行追偿。
#### （2）施工总承包单位不可追偿
施工总承包单位将建设工程分包给个人或者不具备合法经营资格的单位，导致拖欠农民工工资的，施工总承包单位均负有清偿责任（第36条）。
# 三、行政权力对农民工工资支付保障的范围和力度更大
## （一）劳动监察部门的查询调查权
《条例》第40条规定：“人力资源社会保障行政部门在查处拖欠农民工工资案件时，需要依法查询相关单位金融账户和相关当事人拥有房产、车辆等情况的，应当经设区的市级以上地方人民政府人力资源社会保障行政部门负责人批准，有关金融机构和登记部门应当予以配合。”
## （二）劳动监察部门责令支付决定具有强制执行效力
《条例》第42条规定：“人力资源社会保障行政部门作出责令支付被拖欠的农民工工资的决定，相关单位不支付的，可以依法申请人民法院强制执行。”
## （三）对违法行为的行政处罚权
《条例》第六章用了10个条款从各个方面规定了违法行为人的处罚范围和处罚措施。

处罚对象主要是：单位；单位的法定代表人或者主要负责人、直接负责的主管人员和其他直接责任人员；金融监管部门等。
处罚措施主要是：责令改正；罚款；限制施工单位承接新工程、降低资质等级、吊销资质证书；约谈建设单位（特指政府投资项目建设单位）负责人，并作为其业绩考核、薪酬分配、评优评先、职务晋升等的重要依据。
# 裁审要点
1. 注意审查劳动者是否属于《条例》规定的农民工。
2. 农民工向用工单位主张工资的，宜审查用工单位是否存在违法使用个人、不具备合法经营资格的单位或者未依法取得劳务派遣许可证的单位派遣农民工且因此而拖欠工资的情形。
3. 农民工向单位主张工资，单位抗辩没有劳动关系不应当支付的，宜审查该单位是否存在借用资质的情形。
4. 建设工程领域，农民工向建设单位主张工资的，宜审查建设单位是否存在逾期支付工程款，将工程违法发包给无资质主体，建设工程项目是否违反国土空间规划、工程建设等法律法规的情形。
# 实操指引
1. 单位招用农民工的，应当建立农民工工资发放台账制度，工资台账至少保存3年。
2. 建设单位需要向施工单位提供工程款支付担保。
3. 施工总承包单位依照有关规定开设农民工工资专用账户。
4. 施工总承包单位应当按照有关规定存储工资保证金，保证金可以用金融机构保函代替。
5. 农民工遭遇欠付工资纠纷的，可以依法向多个主体主张清偿工资，而不限于直接招用其的个人或者单位。
