合同内容是为合同条款所固定和表现的双方当事人表示一致的内在意思。合同条款表明了合同内容，合同内容在法律文书上表现于条款。《民法典》第668条第2款规定：“借款合同的内容一般包括借款种类、币种、用途、数额、利率、期限和还款方式等条款。”在现实中，由于民间借贷具有自由性的特征，加之有些当事人订立合同行为（包括出具借据、借条）不够规范，许多民间借贷合同条款不具备该条规定的内容，即使具备这些内容也不一定合法。
# 一、绝对必要内容与相对必要内容
除了具备民事行为能力条件外，民间借贷合同应当记载借贷的主要内容才具有法律效力。根据《民法典》有关规定和民间借贷特点来分析，依民间借贷合同记载内容对效力的影响，可分为绝对必要内容与相对必要内容。
## 1.绝对必要内容
绝对必要内容是民间借贷合同上必须记载的内容，如果缺一项则会使整个合同不能成立。民间借贷合同绝对必要内容主要是借贷数额和借款人名称，即合同上的标的额和借款主体，这是民间借贷合同成立的绝对要件。如果没有载明借贷数额，就无法确定标的额，该借贷合同就不能成立，这种情况极为少见，但由于当事人疏忽大意或偶然失误，实践中也出现过。民间借贷合同不仅要有明确、具体的货币数额，而且应当在合同上使用中文数字大写。从实践来看，借款人取款后，往往随手立借据交给出借人，自己很少留底。在这种情况下，如果不使用中文数字大写，而使用阿拉伯数字，因阿拉伯数字容易改动，且有时难以鉴定，就容易发生借贷数额纠纷。发生这类纠纷，借款人往往吃亏，原因是借据在出借人手中，出借人举证充足、有力，而借款人手中往往缺乏证据，容易出现举证不能，而法院审理案件是重证据的，只能依据证据作出判决。

民间借贷合同如果没有借款人签名或盖章，该借款合同因缺乏一方主体就不能成立，出借人持有这种借据，除非借款人承认，如果被指认的借款人不予认可，出借人又无其他证据证明的，该借据对出借人指认的借款人无效。当然，有其他证据证明被指认的借款人就是实际借款人，如通过笔迹鉴定，证明该借据确为被指认的借款人所书写的，可以认定书写人为借款人；书写人能够提供证据证明自己为他人代写，并得到实际借款人承认的，则由实际借款人承担偿还责任。

民间借贷合同上有借款人却无出借人的，一般认定借据持有人为出借人，但借款人或者实际出借人有证据证明借据持有人不是出借人的，该借据对持有人无效。

民间借贷合同缺乏必要内容并非当然无效，当事人双方一致承认并无异议，或者借款人自动履行义务的，则可追认该民间借贷合同自始有效。
## 2.相对必要内容 
相对必要内容是由当事人自由决定是否记载于借贷合同的内容，若记载便有约定效力，若不记载可依法处理而不影响民间借贷合同本身效力。民间借贷合同的相对必要内容主要有用途、利率、期限等。
# 二、借款用途
借款用途是借款方使用借款的范围和内容。金融机构为了有利保证国家产业政策的实施和国民经济的协调发展，以及保证贷款安全，严格规定了贷款用途，并有权监督借款人使用贷款的权利，所以，对银行贷款来说，贷款用途是必备条款。在民间借贷活动中，大多数借款人为了取得借款，往往主动向出借人说明借款用途，希望得到出借人的理解和支持；出借人在出借前，为了保证借款安全，也往往问清借款用途。借款用途是民间借贷当事人是否考虑出借资金的主要条件，但没有约定借贷用途，一般不影响合同的效力和合同规定的权利义务关系。如果已经明确约定借款用途，而借款人不按约定用途使用借款的，则构成违约，出借人可以根据《民法典》第673条的规定，提前收回借款或者解除合同。借款人虚构用途骗取借款，如借款人为了偿还赌债，却以生活困难为由向出借人借款，出借人可以按照《民法典》有关规定，主张民间借贷合同无效或者请求予以撤销，并要求立即返还借款。
# 三、借款利息
对银行贷款来说，利润是其生存发展之本，而利润主要来源于利息，所以，利率或利息不仅是金融借款合同的主要条款，而且是必备条款。但对民间借贷来说，利率或利息不是必备条款，有些出借人不要利息或者合同没有约定利息，这并不影响无息民间借贷合同的法律效力，如果对有无利息产生争议，则应按照《民法典》以及新《最高院民间借贷规定》的有关规定处理。双方当事人已经约定利率或利息的，应当在民间借贷合同中作为条款记载下来，如果不记载在借款合同之中，借款人又予以否认的，往往被认为是无息借款。
# 四、还款方式 
还款方式，是借款人归还借款的结算方式。民间借贷的还款方式由双方当事人自由约定，可以约定一次还款，也可以约定分多次还款，还可以约定其他还款方式。还款方式没有约定或者约定不明确的，不影响整个合同效力，双方当事人可以协议补充；不能达成补充协议的，应当认定一次性以金钱方式偿还。
# 五、借贷期限
民间借贷期限是指借款的使用时间。民间借贷合同明确约定借贷期限，有利于借款人有计划地使用借款和准备还款，对出借人来说有利于按时实现债权，但民间借贷使用期限不是民间借贷合同的必备条款，双方当事人没有约定或者约定不明确的，不影响合同效力。

没有约定期限有两种情况：一是明确不约定期限，如借款人向出借人借款时，询问出借人可以借用多少时间，出借人表示不用约定借用多长时间，需要归还时再告诉借款人，其意思是期限问题留在日后再确定，这是明确不约定期限；二是没有约定期限，双方当事人在发生民间借贷时，没有就期限问题进行协商，结果在合同中没有借款期限条款，在这种情况下，借款使用期限也处于待定状态。

民间借贷合同如果没有约定期限或者约定不明确，则有可能发生期限争议和如何确认还款期限问题。《民法典》第675条规定：“借款人应当按照约定的期限返还借款。对借款期限没有约定或者约定不明确，依据本法第五百一十条的规定仍不能确定的，借款人可以随时返还；贷款人可以催告借款人在合理期限内返还。”根据《民法典》第510条的规定，当事人对借贷期限没有约定或者约定不明确的，可以协议补充；不能达成补充协议的，按照合同有关条款或者交易习惯确定。出借人或者借款人如果不与对方就期限问题进行补充协议；或者无法与对方达成补充协议，不能认为借贷永远处于无期限状态，出借人可以根据《民法典》第510条的规定，催告借款人在合理期限内返还借款。

这里的合理期限，通常是指债权人给予债务人必要的准备履行义务的一段时间。在民间借贷合同未约定履行期限或约定不明的情况下，出借人给借款人合理期限主要表现为催告，至于催告中的“合理期限”是几天或几个月，法律没有量的规定，而应当以合理为原则，由出借人根据借款人的实际情况来决定，借款人不能认为出借人决定的期限不符合客观实际来对抗出借人决定的履行期限，而只能与出借人协商确定还款期限。有些借款人可能在一年两年，甚至更长的时间内没有还款能力，如果把合理期限与借款人有履行还款能力等同起来，则不利于对出借人合法权益的保护。出借人行使催告权时给予借款人的合理期限，能够产生四个法律效果：一是表明出借人已经主张债权，要求借款人偿还借款；二是将产生诉讼时效中断；三是给予合理期限后，借款使用期限就确定下来；四是还款付息履行期产生，借款人合理期限届满时未还款付息的，负有迟延履行责任。履行期限与借款期限不同，借款期限对借款人来说是指使用期限，履行期限是指债务人向债权人履行义务时间。

从司法实践来看，合理期限并非只有债权人才可以决定，当出借债权人在诉讼前未催告而未给合理期限时，民间借贷纠纷案件进入诉讼程序后，法院可以利用司法公权给债务人一个合理期限，如重庆市北碚区人民法院（2020）渝0109民初1671号民事判决书中表述“对于还款的＇合理期限＇，结合日常交易习惯，本院确定为本判决生效之日起十日内”。

此外，司法实践中还有一种观点认为，在认定合理期限的问题上，应将被告债务人收到法院起诉状之日视为债权人向债务人发出的最后一次催告日，并将法院开庭审理日视为宽限期届满日。主要理由是，在民事一审普通程序中，人民法院在立案之日起5日内将起诉状发送被告，被告在收到起诉状之日起15日内提出答辩状，此司法行为与债权人向债务人催告一样具有催告效力，且在15日的答辩期可视为合理期限，在民间借贷合同未约定履行期限或约定不明的情况下，出借人在此期限内履行义务的，亦应视为期内履行，因而不产生违约责任。
# 借贷风险提示
- 民间借贷合同在内容上有可能存在以下几个主要法律风险：
	1. 民间借贷合同没有载明借贷数额，又无证据予以确定的，该合同不能成立，出借人如果提起诉讼，人民法院将驳回其诉讼请求。
	2. 民间借贷合同中的借款数额只使用阿拉伯数字而未使用中文数字大写，阿拉伯数字如果被出借人改动，通过文迹鉴定也不能确定的，借款人将承担少借多还的不利后果。
	3. 民间借贷合同没有借款人签名或盖章，又无其他证据确定借款人的，该合同对出借人指认的所谓借款人无效。
	4. 借款人与出借人口头约定借款用途，但未在借款合同中载明，借款人违约使用借款危及债权安全的，出借人难以主张借款人违约而要求提前偿还借款。
	5. 绝大多数出借人出借资金本意是为了取得利息，但在借款合同中没有载明利息或利率，借款人主张无息的，出借人就不能取得利息，由此造成的利息损失由出借人自行承受。
	6. 民间借贷合同已经明确约定借款期限，借款人逾期偿还借款的，将按约承担支付逾期利息、违约金等责任。
	7. 民间借贷合同没有约定借款期限的，出借人可以随时要求借款人偿还，这会使不少借款人措手不及，故出借人应当给借款人一个合理的偿还时间。
# 案例3-3
2018年3月10日，尤某给张某出具一份借条载明：尤某因经销白酒缺乏资金向张某借款50万元，3年内返还，张某在尤某需要资金时打款至尤某银行账户。2018年6月8日、8月14日，张某两次向尤某给付借款，金额分别为20万元、10万元，共计30万元。嗣后，张某发现尤某未将所借款项用于白酒经销，资金去向不明，遂停止向尤某借款，并向尤某索要借款，但催讨无果。

2018年9月张某向松滋市法院提起民间借贷纠纷诉讼，请求判令被告尤某立即偿还原告借款30万元，并自借款之日起按每月2％利率标准计付利息至清偿之日止。

被告尤某未答辩，也未提交证据。

松滋市法院认为，原告张某与被告尤某之间因提供借款建立民间借贷关系，合法有效，应予维护。原被告之间约定借款期限3年，借款用途为经销白酒，但被告尤某未按照约定的借款用途使用借款资金，资金去向不明，原告张某可以依照合同法规定停止给付借款并提前收回已给付的借款。故原告要求被告立即返还借款的诉讼请求合理合法，予以支持。原、被告之间的借款未约定支付利息，应视为不支付利息，原告主张按月利率2％支付从借款之日起至清偿之日止的利息，没有法律依据，但可按年利率6％的标准主张逾期利息，即被告应承担从起诉之日起至清偿之日止的利息。

松滋市法院判决：被告尤某于本判决生效后5日内返还原告张某借款30万元，并按年利率6％标准支付从2018年9月12日起至清偿之日止的利息。

# 案例3-4
向某某与罗某某原系所谓的姻亲关系。2015年4月，罗某某因资金需要向向某某借款20000元，并出具借条一份，该借条上载明“今借到向某某现金（20000）元整大写（贰万元整）。借款人：罗某某。2015年4月”。2016年3月4日，罗某某再次因资金需要向向某某借款10000元，并出具借条，该借条上载明“今借到向某某现金（10000）元整大写（壹万元整）。借款人：罗某某。2016年3月4日”。上述借款均以现金方式由向某某交付给罗某某之子罗B。罗某某一直尚未向向某某偿还。

2020年3月，向某某向北碚区法院提出民间借贷纠纷诉讼称：向某某与罗某某原系亲家关系，2015年4月至2016年3月罗某某以开办沙场需要资金为由向原告借款30000元。现向某某要求罗某某还款遭到罗某某拒绝，请求法院判令被告罗某某立即偿还原告借款30000元。

被告罗某某辩称：借款30000元属实，现无力偿还，且该笔借款诉讼时效期间已经届满，要求驳回原告诉讼请求。

北碚区法院认为：本案的争议焦点在于双方之间借贷关系的诉讼时效是否已经届满。根据《合同法》第206条规定：“借款人应当按照约定的期限返还借款。对借款期限没有约定或者约定不明确，依照本法第六十一条的规定仍不能确定的，借款人可以随时返还；贷款人可以催告借款人在合理期限内返还。”第61条规定：“合同生效后，当事人就质量、价款或者报酬、履行地点等内容没有约定或者约定不明确的，可以协议补充；不能达成补充协议的，按照合同有关条款或者交易习惯确定。”本案中，罗某某分别于2015年4月及2016年3月4日向向某某出具借条，向某某按照约定以现金方式交付了借款共计30000元。双方未在借款时约定还款期限，事后亦未制订补充协议。故向某某有权随时要求罗某某在合理期限内偿还借款，本案借款的诉讼时效期间并未届满，对罗某某的辩解理由，本院不予采纳。对于还款的“合理期限”，结合日常交易习惯，本院确定为本判决生效之日起10日内。综上所述，对向某某要求罗某某偿还借款30000元的诉讼请求，本院予以支持。

北碚区法院判决：被告罗某某在本判决生效之日起10日内向原告向某某偿还借款30000元。

［根据重庆市北碚区人民法院（2020）渝0109民初1671号民事判决书编写］
# 案例3-5
葛某（男）与任某（女）曾系男女朋友关系。2016年10月至2018年12月31日，葛某使用任某的信用卡、蚂蚁借呗等进行消费。2019年3月12日，葛某向出具《借条》一张交给任某，该借条载明：本人葛某刷任某信用卡本金12万元，利息以银行账单为准，债务人于2019年3月31日前将本金和利息全部还清，特立此据，借条为证。借款人葛某。

2019年12月10日，任某通过手机短信形式给葛某发送短信，明确截至2019年12月7日欠银行总金额为140，529.1元。葛某回复短信：“总数是140，530元，反正我还一笔就少一笔嘛；如果因为倒卡产生的费用，和之前差你的，都会转给你。”

2019年4月29日，任某向江北区法院提起民间借贷纠纷诉讼，称：2016年10月至2018年12月31日，葛某因缺钱，在任某不知情的情况下，私自使用任某的信用卡进行消费，后经双方结算后，葛某出具《借条》一张，确认了尚欠任某借款的金额，被告葛某在2019年12月10日之后归还借款本金7250元，剩余借款本金133，279.1元一直未还本付息。请求：（1）判令被告葛某向原告任某偿还借款本金133，279.1元；（2）判令被告葛某支付原告任某自2020年4月29日起至付清之日止的利息（按年利率6％计算，利随本清）。

在诉讼中，因葛某下落不明，江北区法院向其公告送达了起诉状副本等应诉材料，公告期届满，被告葛某未答辩。
江北区法院认为，葛某向任某出具的借条、葛某与任某的短信聊天记录明确了双方借贷关系、借款金额以及借款已实际支付的事实，双方借款合同关系成立并生效。双方未约定还款期限，任某以诉讼方式要求葛某还款，相关意思表示于本院公告期满之日视为到达葛某，至今已有一月有余，应当认定已满足法律“合理期限”之规定，故对任某要求葛某返还借款本金的诉讼请求，本院予以支持。双方未约定支付利息，应视为不支付利息，但任某要求葛某自2020年4月29日起支付逾期利息的诉讼请求，符合法律规定，本院予以支持。

江北区法院判决：被告葛某于本判决生效之日起10日内返还原告任某借款本金133，279.1元；被告葛某于本判决生效之日起10日内支付原告任某逾期利息（以133，279.1为基数，自2020年4月29日起至付清之日止，按年利率6％计算）。
